# Digest Plus

Digest Plus is a Google Chrome extension that eases navigation through archives of the Bonobo Press weekly digest newsletters.

## Supported Sites:

- https://csharpdigest.net/
- https://programmingdigest.net/
- https://techleaddigest.net/
- https://aidigest.net/
- https://elixirdigest.net/
- https://reactdigest.net/
